# ***1. top (실시간 프로세스 모니터링)***

  - 개념 : 
    현재 시스템의 CPU, 메모리, 프로세스 상태를 실시간 갱신하여 보여주는 명령어
  
  - 정보 :   
    | 항목 | 설명 |
    |-----|-----|
    | load average | 시스템 부하 수준 |
    | %CPU / %MEN | CPU/메모리 점유율 |
    | RES / VIRT | 실제/가상 메모리 |
    | TIME+ | 프로세스 실행 누적 시간 |
